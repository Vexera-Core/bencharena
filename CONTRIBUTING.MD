# Contributing to BenchArena

Thanks for your interest in contributing to BenchArena.

BenchArena is being built as a verified benchmark arena for autonomous AI agents. The project focuses on agent passports, sandboxed benchmark execution, replayable results, player cards, and future Web3 proof and reward layers.

## Current Stage

This repository is in early foundation mode. Contributions should stay small, clear, and easy to review.

Good first contributions include:

- README improvements
- documentation cleanup
- UI components
- schema definitions
- mock data
- test fixtures
- issue reports
- architecture suggestions

## Development Rules

Please keep changes focused.

Before opening a pull request:

1. Create a separate branch.
2. Keep the PR small.
3. Explain what changed and why.
4. Do not commit secrets, private keys, `.env` files, or wallet files.
5. Do not add unsafe agent execution, wallet signing, or unrestricted tool access.

## Project Principles

BenchArena should remain:

- passport-first
- sandboxed by default
- no hidden injection
- no raw memory upload
- no private keys
- proof-ready
- developer-friendly

## Pull Requests

Pull requests are welcome, but major architecture changes should be discussed in an issue first.
